## 🔧 Common Mac Issues & Fixes

### ❌ "The extension needs to be approved" / Driver won't load

**Fix for Intel Macs:**
1. Go to **System Preferences → Security & Privacy → General**
2. Look for a message about a system extension from "Paragon Software"
3. Click **Allow** 

**Fix for Apple Silicon (M1/M2/M3/M4) Macs:**
1. Shut down completely
2. Press and hold power button until **Options** appears
3. Click **Options → Continue**
4. Go to **Utilities → Startup Security Utility**
5. Set Security Policy to **Reduced Security**
6. Check **Allow user management of kernel extensions from identified developers** 
7. Reboot

---

### ❌ "Paragon NTFS for Mac doesn't see my drive"

**Fix:**
- Ensure the drive is properly connected (try a different port/cable)
- Check if the drive appears in **Disk Utility**
- If the drive is NTFS-formatted, try a different USB port
- Restart your Mac with the drive connected

---

### ❌ "Full Disk Access not working"

**Fix (macOS Ventura/Sonoma/Sequoia):**
1. Open **System Settings → Privacy & Security → Full Disk Access**
2. Click the **+** button (may need to unlock first)
3. Navigate to `/Library/Application Support/Paragon Software/`
4. Select `com.paragon-software.ntfsd`
5. Toggle the switch to enable it 

---

### ❌ "My license key isn't working"

**Fix:**
- Verify your license key format matches your product version :
  - **NTFS for Mac 17.x**: Activated via Paragon Licensing Center (LC) with email/password
  - **NTFS for Mac 15.x**: Activates with product key (PSG-3xxx format in ParagonBox)
  - **NTFS for Mac 14.x**: Activates via MyParagon portal (PSG-6xx format)

**Check your version:**
1. Open Paragon NTFS for Mac
2. Click menu **Paragon NTFS → About Paragon NTFS**
3. Note the version number
4. Go to the [correct license portal](https://ntfs-mac.github.io/paragon-ntfs/) for your version 

---

### ❌ "After macOS update, my NTFS drive is read-only again"

**Fix:** Major macOS updates can temporarily disable kernel extensions.
1. Reboot your Mac
2. If that doesn't work, reinstall Paragon NTFS for Mac
3. Version 17+ auto-updates to handle this automatically
